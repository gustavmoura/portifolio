# ✍🏻 Como instalar

Clone o projeto, utilizando o terminal:

```bash
  git clone https://github.com/guioliveirx/guioliveirx-portfolio.git
```

Em seguida:

```bash
  npm install
```

Para rodar o projeto:

```bash
  npm run dev
```
